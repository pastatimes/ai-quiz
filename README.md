# Gebruik jij AI, of gebruikt AI jou?

Een gratis quiz van vijf minuten die laat zien waar je staat als bewuste AI-gebruiker: wat deel je (onbewust) aan data, en waar kun je bijsturen?

**Doe de quiz:** [pastatimes.github.io/ai-quiz](https://pastatimes.github.io/ai-quiz/)

## Stack

Statische web-app (PWA), quiz-logica volledig in de browser, nieuwsbriefkoppeling via MailerLite. Geen tracking.
